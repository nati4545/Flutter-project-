# Flutter Navigator 2.0 Demos

This repository contains three demo projects for Flutter go_router / Navigator 2.0:

- `go_routing_demo01`
- `go_routing_demo02`
- `go_routing_demo03`

## Screenshots

All screenshots are collected in `SCREENSHOTS.md`.

## Run each demo

From the repository root:

```bash
cd go_routing_demo01
flutter run -d chrome
```

Repeat for `go_routing_demo02` and `go_routing_demo03`.
